Soy Desarrollador Web con experiencia en Frontend y Backend, especializado en la creación de interfaces modernas, intuitivas y responsive utilizando tecnologías como React.js, CSS3 y JavaScript. Actualmente, estoy ampliando mi perfil hacia el desarrollo Full Stack, trabajando con herramientas del ecosistema Node.js como Express, MongoDB, Mongoose, Multer, motores de plantillas, WebSockets, y sistemas de autenticación mediante Passport (JWT), sesiones y cookies. También he integrado servicios de mensajería como Twilio y Nodemailer, y aplico buenas prácticas de desarrollo como la arquitectura por capas, uso de DAO, DTO, controladores y routers personalizados para asegurar proyectos escalables, organizados y mantenibles. Realizo testing y documentación de APIs utilizando Swagger y manejo de docker. Además, cuento con experiencia en SQL y bases de datos relacionales, donde diseñé e implementé un sistema completo para un e-commerce. En este proyecto puse en práctica el modelado y normalización de datos, la creación de tablas con relaciones mediante claves primarias y foráneas, y el desarrollo de consultas avanzadas para obtener reportes de ventas, carritos activos y productos con bajo stock. Asimismo, implementé vistas, funciones y procedimientos almacenados para automatizar procesos, triggers para la actualización del stock en tiempo real y transacciones que garantizaron la integridad de las operaciones críticas.
